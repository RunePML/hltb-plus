# hltb-plus
A userscripts for HowLongToBeat with some QoL improvements

## Features list:
- Keeps __Progerss Timer__ visible on *mobile/small* devices.
- Adds a "Export" button to the user Games list to export the displayed list to a CSV file.
- Displays progress on the page title when a timer is running.
- Confirmation dialog to avoid losing progress on a running timer when __reloading__ or __closing__ the tab.
